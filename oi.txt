bibliotecas front end(framework)
opções: 	python: django 
		Js: React 


api uber(python) ->django ou Flasky:
		Solicitar uma viagem: É possível solicitar uma viagem para um local específico usando a API da Uber.

		Obter detalhes da viagem: É possível obter informações detalhadas sobre uma viagem, como o motorista atribuído, a rota e o preço estimado.

		Obter detalhes do perfil do usuário: É possível acessar informações do perfil do usuário, como nome, foto e endereço de e-mail.

		Gerenciar pagamentos: É possível realizar pagamentos dentro do aplicativo usando os métodos de pagamento da Uber.

		Listar produtos disponíveis: É possível obter uma lista dos produtos (categorias de veículos) disponíveis na área atual.

		Obter estimativas de preço: É possível obter estimativas de preço para uma viagem com base na localização inicial e final.

		Obter histórico de viagens: É possível acessar o histórico de viagens do usuário, incluindo detalhes como data, hora, local de partida e destino.


api skyscanner:
		Pesquisar voos: Você pode fazer consultas de pesquisa de voos, especificando a origem, destino, datas de partida e retorno, número de passageiros e outras opções relevantes. A API retornará resultados com detalhes sobre voos disponíveis.

		Obter preços: Você pode obter informações de preços de voos para rotas específicas. Isso permite que você encontre as melhores tarifas disponíveis para determinados destinos em datas específicas.

		Verificar a disponibilidade: Através da API, é possível verificar a disponibilidade de voos em determinados horários e datas. Isso pode ser útil para planejar viagens com base na disponibilidade de voos.

		Informações sobre companhias aéreas: Você pode acessar informações sobre diferentes companhias aéreas, como nome da companhia, detalhes de contato e políticas específicas da empresa.

		Configurar alertas de preços: A API Skyscanner permite configurar alertas de preços, para que você seja notificado quando os preços dos voos para um determinado destino atingirem um valor desejado.

		Acesso a detalhes do voo: Você pode obter informações detalhadas sobre um voo específico, incluindo horários de partida e chegada, duração do voo, número do voo, entre outros.
